# The Adapterbook Package

AdapterBook offers a streamlined environment for testing component adapters with an integrated JSON editor for real-time property modification. Developers can instantly preview how components respond to different prop values through its intuitive interface. This tool eliminates the need to modify source code for testing, significantly accelerating the development workflow and ensuring component quality through efficient, centralized testing.

## How to Start

```bash
// Start the vite development server (Adapterbook)
bun run dev
```

or use the alias (If you work in our dev-container environment)

```bash
adapterbook
```
