# Next.js App Stellenportal - Individual job-portals for customers of our HR software

This Next.js App works as job portals for our customers. Each customer can have their own job portal with this nextjs app.

## Technloogies used

**Framework**: Next.js 16.0.10 with App Router
**Internationalization**: next-intl for localization
**Development**: Custom dev servers with HTTPS support
**Styling Integration**: Emotion compiler integration
**Other Packages**: This package uses other "sibling" packages from this monorepo. For example sui-core - a component library.
**Docker**: For deployment we use docker. We create a docker container and deploy this container to AWS ECR
