# @hrworks/sui-core

This package is the core component library for HR WORKS

## Technology Stack

- **React:** 19
- **Styling:** emotion with styled components
- **Design System:** The components are built on top of our HR WORKS design system and use design tokens

## Installation

To use the components from this package, install it via npm, pnpm or bun:

```bash
npm add @hrworks/sui-core
pnpm add @hrworks/sui-core
bun add @hrworks/sui-core
```

## Usage

Import components and use them in your React application:

```jsx
import { Button } from '@hrworks/sui-core';

function MyApp() {
  return <Button>Click me</Button>;
}
```
